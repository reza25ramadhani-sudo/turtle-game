# Python Turtle Race Simulation

An event-driven, object-oriented simulation application built in Python using the Turtle graphics library. This project focuses on managing multiple object instances simultaneously and handling dynamic program states based on user input.

## Features
* **User Interaction:** Prompts the user to place a bet on a specific colored turtle via a graphical text input window.
* **Instance Management:** Dynamically instantiates and manages six distinct turtle objects with independent coordinate tracking.
* **Randomized State Execution:** Implements a randomized coordinate stepping system to simulate a fair, real-time racing environment.
* **Conditional Win Logic:** Evaluates finishing line thresholds and triggers a graphical/console output confirming if the user's prediction was correct.

## Core Concepts Applied
* Object-Oriented Programming (OOP)
* Event-Driven Programming
* Coordinate Systems & Screen State Management
